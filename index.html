Before We Begin

What is JavaScript?

JavaScript is a programming language used to make websites interactive. It allows websites to respond to user actions, perform calculations, validate forms, fetch data, build games, and much more.

How to Open the Browser Console

  • Press F12
  • Go to the Console tab
  • Type JavaScript code there

2. Variables

Variables are containers used to store data.

let name = "Ali";

let age = 20;

console.log(name);
console.log(age);

3. Variable Naming Rules

Valid:

let firstName;
let age1;
let _score;
let $price;

Invalid:

let 1name;
let user-name;
let let;
Use camelCase naming: 
let totalMarks;
let userAge;
let phoneNumber;

4. var vs let vs const

var

var city = "Lahore";

Old way. Avoid in modern JavaScript.

let

let age = 20;

age = 21;

Use when value can change.

const

const pi = 3.14159;

Use when value should not change.

Modern Recommendation:
  • Prefer const by default.
  • Use let if reassignment is needed.
  • Avoid var.

3. == vs ===

5 == "5";   // true

5 === "5";  // false
== checks value. === checks value and data type. Professionals usually prefer ===.

5. if Statement

6. if...else

7. else if

    console.log("A+");
}

    console.log("A");

9. switch Statement

let day = 2;

switch(day){

    case 1:
        console.log("Monday");
        break;

    case 2:
        console.log("Tuesday");
        break;

    default:
        console.log("Invalid Day");
}

10. Ternary Operator

let result = age >= 18

Mini Project — Console Choice Game

let choice = prompt(
"Choose:\n1. Tea\n2. Coffee\n3. Juice"
);

if(choice === "1"){
    console.log("You chose Tea.");
}
else if(choice === "2"){
    console.log("You chose Coffee.");
}
else if(choice === "3"){
    console.log("You chose Juice.");
}
else{
    console.log("Invalid Choice.");
}
